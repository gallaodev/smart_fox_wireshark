Você usa o wireshark? Ja teve dificuldades para montar um filtro? Sabia que é possivel montar gráficos no wireshark? Abaixo algumas dicas que podem te ajudar nestes assuntos...


É normal ter algumas dificuldades quando você começa a usar o wireshark. Ler um arquivo pcap de uma captura ou mesmo capturar em tempo real os pacotes é o minimo que você deve conseguir fazer para começar a interpretar os dados, acredito que poucos tem dificuldades com isso.
O próximo passo é fazer alguns filtros para localizar pacotes especificos para sua análise, filtrar um protocolo e um ip é relativamente simples, mas em poucos minutos voce descobre que isso não ajuda muito no caso de capturas um pouco mais longas, dai surge a necessidade de filtros mais especificos...


Antes de filtrar é importante saber como o filtro funciona.


O Wireshark usa a tecnologia BPF - Berkeley Packet Filter e os campos de filtros podem ser divididos em:


  • Identifiers - Dados que representam as camadas TCP/IP e os seus protocolos.
  • Qualifiers - Dados que subdivivem as camadas, podem ou não ser usado nos filtros.
    Alguns exemplos são: Type (host, port, net), Direction (src, dst) e Proto (TCP or UDP).
  • Logic Operators - São utilizados para definir as regras do filtro com realção aos values.
  • Values - Valores que você deseja filtrar, podem ou nao ser usados nos filtros.

Exemplos de filtros


A montagem dos filtros é feita pela seguinte syntax: [identifiers][.][qualifiers][logic operators][value], o wireshark faz um check na syntax e coloca uma cor de fundo no campo de syntax para te auxliar, sendo verde para ok e vermelhor para nok.


  • Filtrando um ip de origem com valor 10.0.0.1: ip.src == 10.0.0.1
  • Filtrando um ip com protocolo na porta 53: ip.proto == 53, lembrando que para filtrar o protocolo direto sem o IP podemos usar ele como identifier
  • Filtrando o protocolo TCP com um source port específico: tcp.srcport == 1147

São muitas formas de fazer concorda, e mesmo com o wireshark te ajudando a completar os filtros voce ainda pode se perder neles. Sendo assim existe uma forma mais facil ainda de montar os filtros. Na area onde ficam as informações de um pacote em modo texto voce pode escolher um campo, selecionar simplemente clicando com botão esquerdo e clicando com botão direito escolher a opção aplicar um filtro e depois selecionado, pronto basta voce manipular operadores logicos e valor e voce com certeza vai chegar no valor procurado.

É possivel de forma bem simples gerar gráficos no wireshark


A princípio parece estranho utilizar o wireshark para gerar gráficos, mas esta função pode ser extremamente util para enriquecer conteudos como relatorios, por exemplo pos mortem. Este é um exemplo de utilização, existem diversas outras possibilidades, ate mesmo auxiliar em um tshoot, muitas vezes olhar o problema por outro ponto de vista pode ajudar.


Segue abaixo um passo a passo para criação:


  1. Escolha na barra de ferramentas Estatísticas.
  2. Escolha a opção Gráficos I/O
  3. Vai aparece uma caixa para edição de graficos
  4. Crie linhas para o seu gráfico indicando cores e filtros (ja mencionei acima como fazer), cada filtro ira representar uma linha com uma cor diferente no seu grafico
  5. Use sua criatividade para criar diferentes graficos e mostrar de forma mais simples uma ideia, teoria ou fato!
